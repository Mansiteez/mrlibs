# mrlibs



## Overview
mrlibs is an R package that enables the user to do find the differentially expressed genes between specified classes. It can be extensively used as a single step exploratory package to provide with the most significant DEGs for a particular case.
## Usage
mrlibs asks for two input variables. First is the data_file which is the path to the data file containing gene expression levels with columns containing gene names. The second is the dataID_file which specifies the path to the dataID file containg columns as SampleId and SubType.
